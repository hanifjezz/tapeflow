About

Tapeflow adalah website sederhana untuk memantau suhu proses pembuatan tape/tapai.

  • Frontend: HTML + CSS + Vanilla JS
  • Backend: Node.js (Express) + MySQL
  • Endpoint: /api/devices, /api/latest, /api/measurements

Versi awal — hanya viewer tanpa login.

Tujuan

Memberikan pemantauan suhu real-time yang mudah dibaca dan stabil untuk proses fermentasi.

Teknologi

ESP32 + sensor suhu → REST API → database MySQL → UI ringan tanpa framework.

Roadmap
  • Threshold & notifikasi
  • Export CSV
  • Multi-batch tracking
